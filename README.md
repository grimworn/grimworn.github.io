# Grimworn Kreatif Stüdyo - React

Reklam animasyonları ve dijital tasarımda sektörün en inovatif çözümlerini sunan modern React web sitesi.

## 🚀 Özellikler

- **React 18** - Modern JavaScript kütüphanesi
- **Framer Motion** - Profesyonel animasyonlar
- **Tailwind CSS** - Hızlı ve esnek styling
- **Responsive Design** - Mobil, tablet ve desktop uyumlu
- **Performance Optimized** - Vite ile hızlı build

## 📦 Kurulum

```bash
npm install
```

## 🔧 Geliştirme

```bash
npm run dev
```

Siteniz `http://localhost:5173` adresinde açılacak

## 🏗️ Build

```bash
npm run build
```

Optimized production dosyaları `dist/` klasörüne kaydedilecek

## 📁 Proje Yapısı

```
.
├── src/
│   ├── components/
│   │   ├── Navigation.jsx    # Navigasyon menüsü
│   │   ├── Hero.jsx          # Ana giriş bölümü
│   │   ├── Portfolio.jsx     # Projeler galerisi
│   │   ├── Services.jsx      # Hizmetler
│   │   ├── Contact.jsx       # İletişim formu
│   │   └── Footer.jsx        # Alt bilgi
│   ├── App.jsx               # Ana uygulama
│   ├── main.jsx              # Entry point
│   └── index.css             # Global stiller
├── package.json              # Bağımlılıklar
├── tailwind.config.js        # Tailwind yapılandırması
├── vite.config.js            # Vite yapılandırması
└── index.html                # HTML şablonu
```
